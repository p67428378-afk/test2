# Portfolio Risk Assessment Microservice

This project is a Java and HTML/CSS implementation of the Portfolio Risk Assessment Microservice.

## Application Architecture

- **Backend**: Java with Spring Boot
- **Frontend**: HTML, CSS, and JavaScript

## Project Structure

- `backend/`: Contains the Spring Boot application.
  - `src/main/java/com/example/riskassessment/`: Main application source code.
    - `controller/`: REST controllers.
    - `model/`: Data models.
    - `repository/`: Data repositories.
    - `service/`: Business logic.
  - `pom.xml`: Maven project configuration.
- `frontend/`: Contains the HTML, CSS, and JavaScript files.
  - `index.html`: The main HTML file.
  - `style.css`: CSS for styling.
  - `script.js`: JavaScript for application logic.

## Setup Instructions

### Backend

1. Navigate to the `backend` directory.
2. Run `mvn spring-boot:run` to start the backend server.

### Frontend

1. Open the `frontend/index.html` file in your browser.
