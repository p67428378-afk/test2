# Health Insurance Management Portal

This project is a full-stack web application that allows policyholders to manage their health insurance policies. Users can view their current policy, update their information, and cancel their policy.

## Application Architecture

The application is built with a modern tech stack:

- **Backend:** FastAPI (Python)
- **Frontend:** React (Vite)
- **Database:** PostgreSQL (using SQLAlchemy)

### Backend Architecture

The backend is built using FastAPI and follows a standard microservice architecture. It includes the following components:

- **`main.py`**: The main application file that initializes the FastAPI app and includes the API routers.
- **`api/`**: Contains the API endpoints, organized by version.
- **`core/`**: Contains the core application settings and configuration.
- **`db/`**: Contains the database session management and base classes for models.
- **`models/`**: Contains the SQLAlchemy models for the database schema.
- **`schemas/`**: Contains the Pydantic schemas for data validation and serialization.
- **`services/`**: Contains the business logic for the application.
- **`tests/`**: Contains the unit and integration tests for the backend.

### Frontend Architecture

The frontend is a single-page application (SPA) built with React and Vite. It includes the following components:

- **`src/components/`**: Reusable UI components.
- **`src/pages/`**: Page components that represent different views of the application.
- **`src/services/`**: Functions for making API requests to the backend.

## Setup Instructions

### Backend

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Create a virtual environment:
    ```bash
    python -m venv venv
    ```
3.  Activate the virtual environment:
    -   **Windows:**
        ```bash
        venv\Scripts\activate
        ```
    -   **macOS/Linux:**
        ```bash
        source venv/bin/activate
        ```
4.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
5.  Run the application:
    ```bash
    uvicorn app.main:app --reload
    ```

### Frontend

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install the required dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## API Documentation

The API documentation is automatically generated by FastAPI and is available at `/docs` when the backend is running.

### Key Endpoints

- **`GET /api/v1/policies/`**: Get a list of all policies.
- **`POST /api/v1/policies/`**: Create a new policy.
- **`GET /api/v1/policies/{policy_id}`**: Get a specific policy by ID.
- **`PUT /api/v1/policies/{policy_id}`**: Update a policy.
- **`DELETE /api/v1/policies/{policy_id}`**: Delete a policy.
